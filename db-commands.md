#DB Commands

- [Create](#create)
- [Empty](#empty)
- [Init](#init)

<a name="create"></a>
##Create

This will try to create the database.

Usage:

	php console db:create

<a name="empty"></a>
##Empty

Empty the database.

Usage:

	php console db:empty

<a name="init"></a>
##Init

Initialize the database configuration.

Usage:

	php console db:init
