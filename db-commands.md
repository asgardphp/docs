#DB Commands

- [Dump](#dump)
- [Empty](#empty)
- [Init](#init)
- [Restore](#restore)

<a name="dump"></a>
##Dump

Dump the MySQL database into a file.

Usage:

	php console db:dump [destination]

<a name="empty"></a>
##Empty

Empty the database.

Usage:

	php console db:empty

<a name="init"></a>
##Init

Initialize the database configuration.

Usage:

	php console db:init

<a name="restore"></a>
##Restore

Restore the database from an SQL file.

Usage:

	php console db:restore [source]
